Photos and Text is run by me, Jared Haight. You can find out a little bit more about me here.


This website was my excuse to teach myself some webdesign, Django and Python. The 1140 Grid CSS System is keeping things together and any other design was achieved through trial and error. All of this is being served out by Nginx and Gunicorn, running on a big scary computer located somehwere in Virginia.


I didn't want to learn javascript, so I took a crash course in jquery instead. It's handling the visual effects, pop-ups and keyboard navigation (you can use left, right, j and k on your keyboard to navigate pages on the site).
